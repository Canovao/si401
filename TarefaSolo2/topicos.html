Tópicos que serão tratados

  • Unidade 1: Visão geral sobre o funcionamento da web, com foco nas interações navegador-servidor, e contextualização da Programação para Web.
  • Unidade 2: Hypertext Markup Language (HTML) e estrutura de documentos HTML (tags, declarações DOCTYPE, elementos HTML e atributos de elementos HTML).
  • Unidade 3: Exemplos de elementos HTML (parágrafos, títulos, imagens, hyperlinks, tabelas, listas e iFrames).
  • Unidade 4: O elemento HEAD de HTML.
  • Unidade 5: Layouts via HTML.
  • Unidade 6: Cascade Style Sheets (CSS): definições, sintaxe e formas de utilização.
  • Unidade 7: Cascade Style Sheets (CSS): propriedades de fundo de tela, de texto e de fonte.
  • Unidade 8: Cascade Style Sheets (CSS): propriedades abreviadas e o modelo de caixas (box model).
  • Unidade 9: Programação no front-end com JavaScript: definições, sintaxe, instruções e o objeto DOM.
  • Unidade 10: Detecção e reação a eventos via JavaScript.
  • Unidade 11: Manipulação de formulários HTML via JavaScript.
  • Unidade 12: Orientação a objetos em JavaScript.
  • Unidade 13: Pogramação no back-end com PHP: infraestrutura necessária e aspectos gerais da linguagem.
  • Unidade 14: Aspectos específicos da linguagem PHP: operadores, estruturas de controle e funções.
  • Unidade 15: PHP e formulários HTML, Server Side Includes.
  • Unidade 16: Orientação a objetos em PHP.
  • Unidade 17: Acesso a Bancos de Dados em PHP via PHP Data Objects (PDO).
  • Unidade 18: Cookies e sessões em PHP.
  • Unidade 19: Comunicação assíncrona de dados entre back-end (aplicações em PHP) e front-end (scripts em JavaScript).
